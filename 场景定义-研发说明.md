## Sheet2｜数据来源清单

| 数据对象 | 来源类型 | 存储 Key / 常量名 | 读写 | 字段/结构摘要 | 使用页面 |
|----------|----------|-------------------|------|---------------|----------|
| 预置业务场景 | 前端常量 | `BP_PRESET_TEMPLATES` | 只读 | id, name, description, intro, devApproach, tags[], groups[] | 列表页、新建流程弹窗 |
| 预置场景分组 | 前端常量 | `groups[].title/leafId/operations` | 只读 | 大模块标题 + leafId + 操作列表 | 卡片示意图、进入工作台 |
| 预置卡片元数据 | 前端常量 | `PRESET_CARD_META` | 只读 | sketchCap, groupIcons, bullets, illu | 列表预置卡片 |
| 操作模块目录 | 前端常量 | `OP_TEMPLATE_MODULE_CATALOG` | 只读 | l1 中心 → l2 大目录 → operations[] | 编辑页左栏 |
| 导航树 | 前端常量 | `navTree` | 只读 | 与 leafId 对齐的菜单结构 | 分组标题解析、工作台跳转 |
| 自定义场景 | 浏览器本地 | `bp_custom_flow_templates_v1` | 读写 | CustomTemplate[] | 列表、编辑、新建流程 |
| 当前用户（原型） | 硬编码 | `getCurrentTemplateUser()` →「刘工程师」 | 只读 | creator 字段默认值 | 保存自定义场景 |
| 业务流程 | 浏览器本地 | `pm_business_processes_v2` | 独立 | 与场景定义无关 | 场景流程菜单 |

### Sheet2-附｜CustomTemplate 字段

| 字段 | 类型 | 必填 | 长度/格式 | 说明 |
|------|------|------|-----------|------|
| id | string | 是 | 新建 `cft_{timestamp}` | 唯一标识 |
| name | string | 是 | max 60 | 场景名称 |
| description | string | 否 | max 240 | 场景描述 |
| published | boolean | 是 | 默认 true | 是否上线 |
| modules | array | 是 | ≥1 项 | 有序操作列表 |
| creator | string | 是 | — | 创建人 |
| createdAt | number | 是 | 时间戳 | 创建时间 |
| updatedAt | number | 是 | 时间戳 | 更新时间 |

### Sheet2-附｜CustomModule 字段

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| id | string | 是 | 目录操作 ID，如 `pm-collect` |
| label | string | 是 | 操作显示名 |
| leafId | string | 是 | 关联导航叶子，如 `physical-modeling` |
| path | string | 是 | 菜单路径，如 `开发中心 / 离线集成 / 离线任务` |
| sectionTitle | string | 是 | 大模块分段标题（通常等于 l2 大目录名） |

### Sheet2-附｜操作模块目录（开发中心·当前结构）

## Sheet3｜交互逻辑与数据限制

### Sheet3-A｜场景定义列表页

| 规则 ID | 交互点 | 触发条件 | 系统行为 | 数据限制/校验 |
|---------|--------|----------|----------|---------------|
| TC-R01 | 进入页面 | 点击「场景定义」 | `goToTemplateCenter()`，Hub 全屏 | — |
| TC-R02 | 全局搜索 | 输入关键词 | 实时过滤预置+自定义卡片 | 匹配 name/description/intro/devApproach/tags，不区分大小写 |
| TC-R03 | 预置·场景流程开发 | 点击按钮 | `startNewFlowFromModuleTemplate(preset)` | 预置场景只读，不可编辑删除 |
| TC-R04 | 自定义·编辑 | 点击「编辑」 | `goToTemplateEditor({ templateId })` | 仅自定义场景 |
| TC-R05 | 自定义·上线/下线 | 点击切换 | `setCustomTemplatePublished` | 即时写 localStorage |
| TC-R06 | 自定义·场景流程开发 | 点击按钮 | `startNewFlowFromCustomTemplate` | **仅 published=true 时显示按钮** |
| TC-R07 | 新增场景 | 点击「+ 新增业务场景」 | 进入空白编辑页 | — |
| TC-R08 | 返回 | 点击「返回场景全景」 | `goToScenePanorama()` | — |

### Sheet3-B｜自定义场景编辑页

| 规则 ID | 交互点 | 触发条件 | 系统行为 | 数据限制/校验 |
|---------|--------|----------|----------|---------------|
| TE-R01 | 追加操作 | 点击左侧 chip | `addOpModule` 追加到右侧 | **同一 leafId 下相同 op.id 不可重复**；已添加 chip 禁用 |
| TE-R02 | 目录搜索 | 输入关键词 | 过滤左栏操作 | 匹配 label、path、一级中心、大目录名 |
| TE-R03 | 大模块上移/下移 | 分组标题栏 ↑↓ | `moveGroup` 整组与相邻大模块换位 | 首组不可上移，末组不可下移 |
| TE-R04 | 大模块删除 | 分组标题栏 × | `removeGroup` 删除该 leafId 下全部操作 | — |
| TE-R05 | 小操作删除 | 操作行 × | `splice` 删除单条 | — |
| TE-R06 | 上线开关 | 点击「下线/上线」胶囊 | `syncPublishCapsule` | 默认「上线」；保存时写入 published |
| TE-R07 | 保存 | 点击「保存模板」 | `saveCustomTemplate` → 回列表 | 见 Sheet3-C 校验表 |
| TE-R08 | 取消/返回 | 取消或顶栏返回 | `goToTemplateCenter()` 不保存 | — |

### Sheet3-C｜保存与上下线校验

| 校验项 | 规则 | 失败提示 |
|--------|------|----------|
| 场景名称 | trim 后非空 | Toast：请填写场景名称 |
| 操作模块 | modules.length ≥ 1 | Toast：请至少选择一个操作模块 |
| 名称长度 | HTML maxlength=60 | 浏览器截断 |
| 描述长度 | HTML maxlength=240 | 浏览器截断 |
| 重复操作 | 同 leafId + 同 op.id | Toast：已在「{大模块}」中添加，不可重复添加 |

### Sheet3-D｜上下线状态矩阵

| published | 列表徽标 | 新建流程弹窗 | 场景流程开发按钮 | 可编辑 |
|-----------|----------|--------------|------------------|--------|
| true | 已上线 | 展示 | 展示 | 是 |
| false | 已下线 | 不展示 | 隐藏 | 是 |

### Sheet3-E｜右侧流程预览分组规则

| 项目 | 规则 |
|------|------|
| 大模块分组键 | `leafId`（同一 leafId 的操作归入同一分段） |
| 大模块标题 | `getTemplateModuleGroupTitle(mod)`：优先 path 第二段（如「离线集成」），否则 navTree 父级名 |
| 小操作展示 | 时间线节点，显示 `label` |
| 步数统计 | 已选 modules 总数（非大模块数） |

### Sheet3-F｜与新建流程 / 工作台联动

| 规则 ID | 说明 |
|---------|------|
| NF-R01 | 新建流程弹窗 = 全部预置 + **已上线**自定义场景 |
| NF-R02 | 进入工作台：`flattenTemplateGroups` → `groupModulesBySection` → 初始化 `devPathFlowState`、`opFlowVisitStack`、`opFlowModuleLeafIds` |
| NF-R03 | 工作台内各阶段可跳跃、可多次创建，**不强制顺序**（弹窗说明文案） |
| NF-R04 | 自定义场景与业务流程实例 **存储 Key 隔离** |

## Sheet4｜用户操作逻辑（按角色路径）

> **流程图版本**：见 [`Sheet4-用户操作流程图.md`](./场景定义-Excel/Sheet4-用户操作流程图.md)  
> **PNG 图片（可直接复制）**：见 [`场景定义-Excel/flows/`](./场景定义-Excel/flows/) 目录，或打开 [`用户操作流程图-预览.html`](./场景定义-Excel/用户操作流程图-预览.html)

### Sheet4-A｜浏览与选用预置场景

| 步骤 | 用户操作 | 系统响应 | 结果状态 |
|------|----------|----------|----------|
| 1 | 点击二级导航「场景定义」 | 打开列表页，展示引导+预置+自定义 | 浏览态 |
| 2 | （可选）输入搜索词 | 过滤卡片 | 浏览态 |
| 3 | 点击预置卡片「场景流程开发」 | 关闭弹窗逻辑不适用；直接进入工作台 | 流程开发态 |
| 4 | 左侧自动选中首模块，右侧展示操作流程分段 | 用户可按场景路径操作 | 流程开发态 |

### Sheet4-B｜创建自定义场景

| 步骤 | 用户操作 | 系统响应 | 结果状态 |
|------|----------|----------|----------|
| 1 | 点击「+ 新增业务场景」 | 进入编辑页（空白） | 编辑态 |
| 2 | 左栏点击操作 chip（如「模型采集」） | 右侧追加；同 leafId 下不可重复 | 编辑态 |
| 3 | （可选）大模块 ↑↓ 调整顺序 | 整组与相邻大模块交换 | 编辑态 |
| 4 | （可选）小操作 × 删除单步 | 移除该操作；左栏 chip 恢复可点 | 编辑态 |
| 5 | 填写名称、描述，选择上线/下线 | 表单赋值 | 编辑态 |
| 6 | 点击「保存模板」 | 校验 → 写 localStorage → 回列表 Toast | 列表态 |
| 7 | （可选）点击「上线」后 | 新建流程弹窗可见该场景 | 可共享态 |

### Sheet4-C｜管理已有自定义场景

| 步骤 | 用户操作 | 系统响应 | 结果状态 |
|------|----------|----------|----------|
| 1 | 列表找到自定义卡片 | 展示状态徽标、创建人、时间、开发思路 | 浏览态 |
| 2 | 点击「编辑」 | 回填名称、描述、模块顺序、上线状态 | 编辑态 |
| 3 | 点击「下线」 | published=false，按钮变为「上线」 | 已下线 |
| 4 | 点击「上线」 | published=true，显示「场景流程开发」 | 已上线 |
| 5 | 已上线时点击「场景流程开发」 | 同预置场景进入工作台 | 流程开发态 |

### Sheet4-D｜从新建流程选用场景

| 步骤 | 用户操作 | 系统响应 | 结果状态 |
|------|----------|----------|----------|
| 1 | Hub 页点击「新建流程」 | 打开「选择模板」弹窗 | 弹窗态 |
| 2 | 选择预置或已上线自定义卡片 | `startNewFlowFromModuleTemplate` | 流程开发态 |
| 3 | 关闭弹窗，进入工作台 | 操作流程按场景分组展示 | 流程开发态 |
